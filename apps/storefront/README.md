# storefront

## Назначение
`apps/storefront` — модуль/приложение RusToK. Здесь находится его код и корневая документация.

## Взаимодействие
- apps/server (API)
- crates/rustok-commerce и crates/rustok-content через backend
- apps/admin (общая экосистема UI)
