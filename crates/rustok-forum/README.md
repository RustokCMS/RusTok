# rustok-forum

## Назначение
`rustok-forum` — модуль форума: темы, ответы, модерация и форумные workflow.

## Что делает
- Управляет темами и ответами.
- Публикует события для индексации и уведомлений.
- Обеспечивает локализацию контента через `locale`.

## Как работает (простыми словами)
1. API создает или обновляет тему.
2. Сервис сохраняет данные в БД.
3. Событие отправляется в EventBus.

## Ключевые компоненты
- `dto/` — структуры запросов и ответов (включая `locale`).
- `services/` — бизнес-логика форума.
- `entities/` — модели SeaORM.

## Кому нужен
Форуму, комьюнити-разделам, поддержке и контентным обсуждениям.
